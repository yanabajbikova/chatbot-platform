Платформа для создания чат-ботов для техподдержки.

Веб-платформа для создания и управления чат-ботами технической поддержки с интеграцией с Telegram и подготовкой к интеграции с VK (заглушка).

 
Функциональные возможности:
- Приём сообщений от Telegram-бота
- Поиск ответов в базе знаний
- Административная панель для управления сценариями
- Работа с PostgreSQL
- Контейнеризация с помощью Docker
- Подготовка к интеграции с VK (заглушка)

Используемые технологии:
- Python 3.9
- FastAPI
- PostgreSQL
- SQLAlchemy
- Jinja2
- Telegram Bot API
- Docker / Docker Compose

!!!Перед запуском необходимо создать файл .env. Скопируйте в него текст .env.example, измените данные на свои.

Запуск проекта через Docker:
1. Убедитесь, что установлен Docker и Docker Compose
2. В корне проекта выполните команду:
docker-compose up --build

Доступ к сервисам:
- Backend API: [http://localhost:8000] (Отображается сообщение о запуске сервера)
- Административная панель: [http://localhost:8000/admin]
  
В административной панели доступно:
- создание и редактирование категорий
- управление базой ответов
- управление сценариями чат-бота
Все изменения сохраняются в базе данных PostgreSQL.

Запуск без Docker (локально):
Для локального запуска:
1. Установить зависимости:
pip install -r requirements.txt
2. Запустить сервер:
uvicorn app.main:app --reload

Telegram-бот:
Telegram-бот подключается к backend-приложению и отправляет сообщения пользователей на API.
Алгоритм работы:
1. Пользователь отправляет сообщение боту
2. Бот пересылает запрос в FastAPI
3. Сервер ищет ответ в базе знаний
4. Ответ возвращается пользователю

VK (заглушка):
В проекте предусмотрена заглушка для интеграции с VK:
- определены точки расширения
- логика взаимодействия не реализована полностью
- добавлена для демонстрации архитектуры
