Платформа для создания чат-ботов для техподдержки.

Веб-платформа для создания и управления чат-ботами технической поддержки с интеграцией с Telegram и подготовкой к интеграции с VK (заглушка).

 
Функциональные возможности:
- Приём сообщений от Telegram-бота
- Поиск ответов в базе знаний
- Административная панель для управления сценариями
- Работа с PostgreSQL
- Контейнеризация с помощью Docker
- Подготовка к интеграции с VK (заглушка)

Используемые технологии:
- Python 3.9
- FastAPI
- PostgreSQL
- SQLAlchemy
- Jinja2
- Telegram Bot API
- Docker / Docker Compose
- Uvicorn
  
Клонирование репозитория: git clone https://github.com/yanabajbikova/chatbot-platform.git
cd chatbot-platform

Создание и активация виртуального окружения:
python3 -m venv venv
source venv/bin/activate   #macOS / Linux

Установка зависимостей:
pip install -r requirements.txt

Настройка переменных окружения:
Создайте файл .env в корне проекта, скопировав и отредактировав .env.example.
! Файл .env не должен попадать в GitHub.

Запуск базы данных:
Убедитесь, что PostgreSQL запущен локально и база данных создана:
CREATE DATABASE chatbot_platform;

Запуск веб-сервера:
uvicorn app.main:app --reload

После запуска сервер будет доступен по адресу:
API: http://localhost:8000
ПАанель администрирования (веб-редактор): http://localhost:8000/admin
Документация API (Swagger): http://localhost:8000/docs

Запуск Telegram-бота:
В отдельном терминале, с активированным виртуальным окружением:
python telegram_bot.py
Если запуск успешен, в консоли появится сообщение:
Telegram-бот запущен

Остановка работы:
Если сервер запущен в терминале:
CTRL + C
Если бот запущен командой:
CTRL + C

Работа с Git:
Проект использует систему контроля версий Git.
Инициализация репозитория:
git init
Добавление файлов в репозиторий:
git add .
git commit -m "Initial commit"
